Summary statistics

The first tab in this view shows summary statistics about the values of the currently selected attribute.

For attributes with a nominal scale, the following parameters will be calculated:

  1. Mode.

For attributes with an ordinal scale, the following additional parameters will be calculated:

  1. Median.
  2. Minimum.
  3. Maximum.

For attributes with an interval scale, the following additional parameters will be calculated:

  1. Arithmetic mean.
  2. Sample variance.
  3. Population variance.
  4. Standard deviation.
  5. Range.
  6. Kurtosis.

For attributes with a ratio scale, the following additional parameters will be calculated:

  1. Geometric mean.

Note: The statistics are calculated using listwise deletion, which is a method for handling missing data. In this method, an entire record is excluded from analysis if any single value is missing. This behaviour can be deactivated in the project settings.
