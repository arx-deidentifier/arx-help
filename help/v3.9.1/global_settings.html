Project settings

The settings window is accessible from either the application menu or the application toolbar.

The Project section can be used to alter general project properties:
  • Project meta data, including name, description and localization.
  • The syntax for CSV import and export.
The Internals section can be used to adjust performance and visualization parameters:
  • Snapshot settings control the space-time trade-off during data anonymization. Larger (relative) snapshot sizes will typically reduce execution time but increase memory consumption.
  • Some analyses and visualizations can be disabled when very large datasets are handled.
The Transformation section provides options for altering and simplifying the presentation of transformed data:
  • Record suppression can enabled or disabled for non-anonymous output data.
  • Attribute suppression can be enabled or disabled for sensitive and insensitive attributes.
The Search method section can be used to configure the search strategy and risk based anonymization:
  • An heuristic search using a time limit can be enabled for high-dimensional data.
  • Utility-driven record suppression can be disabled to improve execution times.
The Utility analysis section can be used to refine summary statistics and quality measurements:
  • Summary statistics may be calculated with or without list-wise deletion.
  • The use of functional hierarchies during the anonymization process can be disabled.
  • Parameters impacting the performance of the analysis of the usefulness of output data for classification tasks may be altered.
The Risk analysis section provides configuration options for the integrated solver for non-linear equation systems:
  • Optimization parameters include the total number of iterations, iterations per try and the required accuracy.
The Genetic algorithm section provides configuration options for the genetic algorithm search strategy:
  • Optimization parameters include the total number of iterations and the mutation probability.
The User interface section provides configuration options for the user interface:
  • An option to open the help window as an independent window.
